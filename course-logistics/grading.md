# Grading

The grading consists of individual components and group project components:

* Individual components
  * In-class quizzes (10%)
  * One short essay (10%)
  * Evaluation plan (10%)
* Project components
  * Project proposal (10%)
  * Paper prototype (20%)
  * Final design (20%)
  * Final presentation (10%)
  * Final report (10%)
